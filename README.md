## Installation

```

```

## Usage

```
Usage: pwgen-python [OPTIONS] [PW_LENGTH] [NUM_PW]

Options:
  -N, --num-passwords INTEGER
  -H, --sha1 SHA1
  -S, --seed TEXT
  -h, --help                   Show this message and exit.
```
